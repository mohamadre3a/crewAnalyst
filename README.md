# CrewAnalyst

> See [`outputs/report.md`](outputs/report.md) for an example report generated by this pipeline.

A CrewAI-powered data analysis assistant for CSV files. Drop in a dataset, provide optional context, and a crew of seven specialized agents profiles the data, computes statistics, detects anomalies, finds correlations, generates charts, and assembles a stakeholder-ready markdown/PDF report.

## How It Works

**Step 1 ‚Äî Profile** ¬∑ `profiler_agent` ¬∑ *claude-haiku*

Classifies every column and produces the `DataProfile` that all other agents receive.

| Tool | What it does |
|---|---|
| `inspect_csv` | Loads the CSV; returns row count, column count, dtypes, and a 10-row sample. |
| `get_null_report` | Returns null count and null percentage for every column. |

Output ‚Äî `DataProfile`: per-column `semantic_role` (`identifier`, `categorical`, `numerical`, `datetime`, `target_metric`, `text`), `inferred_domain`, `quality_flags`, `has_datetime_index`, `has_target_metric`.

---

**Steps 2, 3, 4 ‚Äî run in parallel** (each receives only the `DataProfile` as context)

---

**Step 2 ‚Äî Statistician** ¬∑ `statistician_agent` ¬∑ *claude-sonnet*

Computes descriptive statistics, group comparisons, and time trends. Surfaces 3‚Äì5 plain-English key findings.

| Tool | What it does |
|---|---|
| `get_descriptive_stats` | Mean, median, std, skewness, kurtosis, min, max, p25, p75 for numeric columns. |
| `compute_categorical_stats` | Unique count, top value, top frequency, and top-20 value counts for categorical columns. |
| `compute_group_aggregates` | Group size and mean of a numeric metric broken down by a categorical column. |
| `analyze_datetime_column` | Trend direction (`upward` / `downward` / `flat`) of a numeric metric over time, plus start/end timestamps and gap/outlier flags. |

Output ‚Äî `StatsProfile`: numeric stats, categorical stats, group comparisons, time trends, key findings.

---

**Step 3 ‚Äî Anomaly Detection** ¬∑ `anomaly_agent` ¬∑ *claude-haiku*

Cross-references IQR and Z-score ‚Äî anomalies flagged by both are elevated in severity. Interprets every finding in the context of the inferred domain.

| Tool | What it does |
|---|---|
| `detect_outliers_iqr` | Flags values outside Q1 ‚àí 1.5√óIQR / Q3 + 1.5√óIQR per numeric column. |
| `detect_outliers_zscore` | Flags values whose absolute Z-score exceeds 3.0. |
| `detect_rare_categories` | Flags categorical values appearing in fewer than 1% of rows. |

Output ‚Äî `AnomalyReport`: up to 20 anomalies sorted by severity, each with detection method (`iqr` / `zscore` / `both` / `rare_category`), severity (`high` / `medium` / `low`), `is_data_quality_issue` flag, and a one-sentence interpretation. Includes total counts and a summary paragraph.

---

**Step 4 ‚Äî Correlation** ¬∑ `correlation_agent` ¬∑ *claude-haiku*

Finds the strongest relationships between variables. Focuses on the target metric when one exists. Skips identifier columns.

| Tool | What it does |
|---|---|
| `get_pearson_correlation_matrix` | Full Pearson correlation matrix for all numeric columns. |
| `compute_cramers_v` | Cram√©r's V association strength for categorical-to-categorical pairs (0 = none, 1 = perfect). |
| `get_descriptive_stats` | Descriptive stats for numeric columns to support interpretation. |

Output ‚Äî `CorrelationReport`: top correlations ranked by absolute coefficient, target metric correlations, redundant pairs (|r| > 0.95), and a summary paragraph with plain-English explanations.

---

**Step 5 ‚Äî Visualizer** ¬∑ `visualizer_agent` ¬∑ *claude-sonnet*

Generates only charts that illustrate a specific finding from steps 2‚Äì4. Maximum 10 charts.

| Tool | What it does |
|---|---|
| `generate_histogram` | Distribution + KDE curve for a numeric column. |
| `generate_boxplot` | Numeric metric split across category groups. |
| `generate_scatter_plot` | Two numeric columns; optionally colored by a category. Used for strong correlations. |
| `generate_bar_chart` | Top N most frequent values in a categorical column. |
| `generate_correlation_heatmap` | Pearson heatmap for all numeric columns. |
| `generate_time_series_chart` | Metric over time; optionally split into multiple lines by a category. |
| `generate_anomaly_highlight_chart` | Index plot with anomalous rows highlighted in red. |

Output ‚Äî `VizManifest`: chart objects with title, caption, finding illustrated, and file path. PNGs saved to `outputs/charts/`.

---

**Step 6 ‚Äî Synthesizer** ¬∑ `synthesizer_agent` ¬∑ *claude-sonnet*

No tools ‚Äî works entirely from structured context. Distills all prior outputs into what a business stakeholder actually needs.

Output ‚Äî `ExecutiveSynthesis`: ranked key findings (each tagged actionable or not), 2‚Äì3 paragraph executive summary in plain English, limitations, 3 concrete recommended next steps, and an overall data quality assessment.

---

**Step 7 ‚Äî Reporter** ¬∑ `reporter_agent` ¬∑ *claude-haiku*

Composes the full report from all prior outputs and saves it to disk.

| Tool | What it does |
|---|---|
| `write_markdown_report` | Writes the complete report to `outputs/report.md` with inline chart references. |
| `convert_markdown_to_pdf` | Converts the markdown to a styled A4 PDF at `outputs/report.pdf` via WeasyPrint. |

Report sections: Executive Summary ¬∑ Key Findings ¬∑ Dataset Overview ¬∑ Statistical Highlights ¬∑ Anomalies ¬∑ Relationships ¬∑ Visualizations ¬∑ What This Data Cannot Answer ¬∑ Recommended Next Steps ¬∑ Data Quality Assessment.

---

## Setup

```bash
uv sync
cp .env.example .env  # add your ANTHROPIC_API_KEY
```

## Run

**Gradio profiler UI** (profile step only):
```bash
uv run app.py
```

**Full pipeline** (all 7 agents):
```bash
uv run python -c "from crewanalyst.crew import run; run('sample_data/test1.csv')"
```

**Profiler test harness** (no UI):
```bash
uv run python agent_tester.py
```
